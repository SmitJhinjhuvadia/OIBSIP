# 🚀 OIBSIP – Oasis Infobyte Web Development Internship

Welcome to my repository for the **Oasis Infobyte Web Development and Designing Internship Program**. This repository contains all the projects completed as part of **Level 2** of the internship.

Each project focuses on strengthening frontend development skills and improving problem-solving abilities using modern web technologies.

## 📌 Level 2 Projects

### 🧮 Task 1 – Calculator Web Application
A responsive and interactive calculator built using HTML, CSS, and JavaScript that performs basic arithmetic operations with a clean and modern user interface.

**Features:**
- Addition, subtraction, multiplication, and division
- Responsive design
- Interactive button animations
- User-friendly interface

---

### 🌟 Task 2 – Tribute Page
A beautifully designed tribute page dedicated to an inspiring personality, showcasing elegant layouts, typography, and responsive design principles.

**Features:**
- Attractive UI design
- Responsive layout
- Structured content presentation
- Optimized user experience

---

### ✅ Task 3 – To-Do Web Application
A dynamic task management application that helps users organize daily activities efficiently.

**Features:**
- Add, edit, and delete tasks
- Mark tasks as completed
- Separate pending and completed tasks
- Clean and intuitive interface

---

### 🔐 Task 4 – Login Authentication System
A modern authentication interface featuring login and registration functionality with an attractive UI design.

**Features:**
- Login and registration forms
- Input validation
- Responsive design
- Professional authentication interface

---
